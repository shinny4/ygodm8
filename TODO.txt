Overworld scripts will be processed by GAS. It's possible to convert them to C using macros for commands and preproc for strings. 
It's probably not worth it; even if the game is ported to another target, it would most likely be supported by GAS.
E.g.
#define display_portrait(portrait, expression, position) 0x23, 0x34, portrait, expression, position,

const unsigned char gScript_E193[] = {
  display_portrait(PORTRAIT_ISHIZU, 0, 8)
};
